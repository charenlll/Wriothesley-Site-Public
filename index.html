Shortcuts

常用操作，用快捷键完成

截图、翻译、剪贴板保存和短文本窗口都可以通过全局快捷键唤起，也可以在设置中重新录制。

Option + A

截图 OCR

区域截图、保存、复制、文字识别和翻译。

Option + T

快速翻译

翻译当前选中的文字，并以浮窗展示结果。

Option + S

保存剪贴板

快速保存剪贴板中的文件或图片。

Option + W / Q

短文本窗口

打开或隐藏临时短文本窗口，便于反复复制。

Privacy

本地优先，使用更安心

账号数据、计划、聊天和便签默认保存在本机。API Key 只保留在本地配置中。

多账号切换

同一台电脑可多人使用，退出登录后回到登录页。

关键配置本地化

API 地址、Key 和模型配置不随账号上传。

计划数据保护

计划写入带备份恢复，降低本地文件损坏风险。
